Creando una web app que logre modernizar un nicho del ámbito jurídico

Cliente:

FucerNet

UX Sprints

El caso:

Fucer, una fundación enfocada en el ámbito jurídico y registral, tenía el objetivo de sumar a su cartera de ofertas educativas un producto digital B2C que reúna toda la normativa relevante para el ámbito.

Así es como nació FucerNet, una aplicación web con un modelo de suscripción mensual a la que un usuario puede acceder a una base de datos de normativas relevantes del rubro mediante un motor de búsqueda.

Lo que hicimos:

Ideación de producto y estrategia comercial

UX Design

Organización de contenido y copywriting. Wireframing. Prototyping. UI.

Desarrollo

Front end. Back end. Integración con Payment Gateway y modelo de suscripciones. PWA.

Resultados

Una PWA accesible desde cualquier dispositivo desktop y mobile con un sistema de suscripciones mensuales automatizadas y una base de datos de contenidos accesible desde un motor de búsqueda y un listado de contenidos favoritos.
